# Keyboard Shortcuts

<ul>
<h4>File Shortcuts</h4>
<li>
<strong>Open:</strong> <kbd>Ctrl</kbd> + <kbd>o</kbd>
</li>
<li>
<strong>Print:</strong> <kbd>Ctrl</kbd> + <kbd>p</kbd>
</li>
<li>
<strong>Save:</strong> <kbd>Ctrl</kbd> + <kbd>s</kbd>
</li>

<li>

<li>

</li>
<li>
<strong>Hand Tool:</strong> <kbd>h</kbd>

<li>
